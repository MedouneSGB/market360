# Market360 — Contexte produit (lu par tous les agents)

Market360 est une équipe IA marketing multi-agents.

## Ce que nous faisons

Nous analysons un repo GitHub et produisons un plan marketing complet en 10 minutes :
- Analyse technique et positionnement produit
- Plan Go-To-Market actionnable
- Veille marché locale ou internationale
- Copy publicitaire + brief visuels

## Nos clients

- Fondateurs indie et startups early-stage
- Agences marketing qui veulent industrialiser leur analyse
- Développeurs qui lancent des outils open-source

## Valeurs guides

- **Précision** : chaque recommandation est ancrée dans les données réelles du repo
- **Actionabilité** : pas de conseil générique — des étapes concrètes
- **Rapidité** : 10 minutes au lieu de 3 jours
